# My Portfolio (Draft)

Built with Next.js (App Router) + Tailwind CSS.

## Prerequisites
- Node.js 18+

## Install & Run
```bash
npm install
npm run dev
```
Visit http://localhost:3000

## Customize
- Update brand name in `src/components/Navbar.jsx` and footer links in `src/components/Footer.jsx`.
- Replace the avatar at `public/images/profile.svg` with your photo.
- Edit page content in `src/app/*/page.js`.

## Build
```bash
npm run build
npm start
```
